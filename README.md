# Pyside2-template
Pyside2 app developpment template

# Developpment (Windows)
To initialize the development environment run the script called `init_dev_environment.bat` located in the `utils` folder or run the following commands
```
py -m venv venv
venv\Script\activate.bat
pip install -r requirements.txt
```

Then enjoy ! On windows, all tools you'll need to build a standalone app are located in the utils folder.
